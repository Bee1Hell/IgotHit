# I got hit！

A RimWorld mod that adds an **Adrenaline Rush** system. When a humanlike pawn is hit and their pain exceeds 20%, adrenaline kicks in with escalating effects.

## Features
- **5 stages** of adrenaline, from minor (+5% move speed) to extreme (+25% move speed, 80% pain reduction)
- **Affects**: pain, move speed, manipulation, sight, rest rate
- **Configurable** via Mod Settings:
  - Rush duration (0.5h ~ 8h)
  - Cooldown duration (1h ~ 72h)
  - All 5 stages' stat values
  - Colony-only trigger toggle
- **Automatic recovery**: expires after the set duration, with a cooldown preventing re-trigger
- **Real-time stage transitions**: severity adjusts dynamically based on current pain
- **Localized**: English + 简体中文
