# I got hit！ — A RimWorld Combat Reaction Mod

Are you tired of vanilla's bland number-vs-number combat?

This mod delivers a **lightweight combat reaction system** built entirely on vanilla mechanics!

## Features

### Adrenaline Rush
When a pawn sustains enough pain (above 20%), adrenaline floods their system. Pain is dulled, mobility increases, but fine motor control and perception suffer — a double-edged sword that keeps fighters in the battle longer at a cost.

### Suppression
Repeated hits build up suppression, affecting aim time and attack cooldowns. A realistic pressure mechanic that rewards tactical positioning.

### Stagger / Off-Balance / Disabled
Based on hit damage and the target's body size, pawns can be staggered, knocked off-balance, or temporarily disabled — adding meaningful consequences to every hit.

## Fully Customizable

Not satisfied with the default values? Everything is configurable in the Mod Settings:
- Duration and severity scaling for each mechanic
- Per-stage stat modifiers (pain factor, move speed, manipulation, sight, etc.)
- Body size influence on stagger thresholds
- Scope filters: apply effects to hostiles only, colonist-only, or everyone

## Known Issues

- **Character Editor**: May display an error when editing hediffs, but functionality is unaffected.
- **CE (Combat Extended)**: Compatibility is unknown and not planned for testing.

## Recommended Mods

- **Down for me!** — Pawns under Adrenaline Rush are more vulnerable to lethal damage; pairing with a downing mod is highly recommended.

## Installation

1. Download or clone the repository.
2. Place the `IgotHit` folder in your `RimWorld/Mods` directory.
3. Enable the mod in the game's Mod menu.
